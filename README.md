# Fake Booter
 A fake booter to entertain the gallery

![free-booter](https://i.imgur.com/aRSprC8.gif)

## How to install ?

```bash
git clone https://github.com/Sayrix/fake-booter
cd fake-booter
npm i
```

## How to start ?

### On Windows
Launch the start.bat

### On CLI
```bash
node index.js
```
